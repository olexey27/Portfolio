# 💼 Portfolio

Willkommen in meinem Portfolio-Repository! 🚀  
Der Quellcode meiner persönlichen Portfolio-Webseite – gebaut mit modernen Technologien und deployed auf meiner eigenen Domain.

🔗 **Live:** [olexey27.space](https://olexey27.space)

---

## 🌟 Features

- **Web CV** – Interaktiver Lebenslauf mit Erfahrungen, Skills, Sprachen und mehr
- **Certificates** – Alle Zertifikate direkt im Browser via Google Drive Preview
- **Kontaktformular** – Nachrichten landen direkt in meiner Gmail via Resend API
- **Responsives Design** – Optimiert für alle Geräte, vom Smartphone bis zum Desktop
- **Animationen** – Smooth Transitions mit Framer Motion
- **Dark Theme** – Durchgängiges dunkles Design mit grünem Accent (`#00ff99`)

---

## 🚀 Technologien

| Bereich | Technologie |
|---|---|
| Framework | Next.js 15 (App Router) |
| Sprache | TypeScript |
| Styling | Tailwind CSS |
| Animationen | Framer Motion |
| Email API | Resend |
| Icons | React Icons |
| Font | JetBrains Mono |
| Hosting | Vercel |
| Domain | olexey27.space |
